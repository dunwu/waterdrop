      1. 消息的顺序问题
      2. 消息的重复问题

      RocketMQ 快速入门

      RocketMQ 快速入门

      Apache RocketMQ 是一个分布式 MQ 和流处理平台，具有低延迟、高性能和可靠性、万亿级容量和灵活的可扩展性。

      RocketMQ 由阿里巴巴孵化，被捐赠给 Apache，成为 Apache 的顶级项目。
