# Java 流量控制

> 在分布式系统中，服务之间的调用关系错综复杂。当某个服务出现故障或流量激增时，如果不加以控制，故障会迅速蔓延，导致整个系统雪崩。流量控制中间件通过限流、熔断、降级等手段，保障系统在恶劣环境下的可用性。
